Panduan Instalasi
1️⃣
STEP 1
Klik Tombol Install / Download
Untuk Tampermonkey Script: klik tombol Install pada card script yang diinginkan. Tab baru akan terbuka otomatis.

Untuk Chrome Extension: klik Download, lalu extract file .ZIP yang terunduh.
2️⃣
STEP 2
Konfirmasi di Tampermonkey / Chrome
Untuk Script: Tampermonkey akan menampilkan konfirmasi install, klik "Install".

Untuk Extension: Buka chrome://extensions, aktifkan Developer Mode, lalu klik "Load Unpacked" dan pilih folder yang sudah di-extract.
3️⃣
STEP 3
Selesai — Siap Digunakan!
Script atau Extension langsung aktif. Buka LiveChat dan fitur otomatis akan berjalan. Untuk mengatur preferensi, gunakan dashboard masing-masing plugin.
