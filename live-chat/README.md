# RKD Script Suite - Cyberpunk Edition

Premium Userscript Suite for LiveChat automation and visual enhancement.

## Features
- **LAPAK1**: Word Highlighter with draggable dashboard and digital chronometer.
- **LAPAK2**: Neural Shield Anti-Spam protection.
- **LAPAK3**: Manga Wallpaper & Dynamic Queue Indicators.

## Installation
1. Open `lapak-installer.html` in your browser.
2. Ensure Tampermonkey has **"Allow access to file URLs"** enabled.
3. Click **INSTALL** on the desired modules.

## Technical Details
- Built with Vanilla JS and CSS Grid.
- Cyberpunk aesthetic with Glassmorphism and Glitch animations.
- Native Tampermonkey `.user.js` detection.
